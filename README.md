# Frontend Mentor - Weather app 🌤

A responsive weather application that displays current weather, hourly forecast, and 7-day forecast for any city.

The app uses the Open-Meteo API to fetch weather data and city suggestions.

🛠 Technologies

CSS3

JavaScript (Vanilla JS)

Fetch API

Open-Meteo Weather API

✨ Features

Search for any city

City autocomplete suggestions

Current weather conditions

Hourly forecast for selected day

7-day forecast

Switch temperature units (°C / °F)

Switch wind speed units (km/h / mph)

Weather icons based on weather codes

Responsive layout
